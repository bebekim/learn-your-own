# Propagation Rules

Propagate by the narrowest valid audience.

Functional/process lessons may broadcast by capability:

- `function.agent.workflow`
- `function.model-routing`
- `function.vision.extraction`

Business/domain lessons require narrow channels:

- workspace
- repo family
- authority source
- risk class

Record delivery separately from effect. Delivery means a future run saw the
adaptation. Effect evaluation asks whether later outcomes improved, regressed,
or showed the adaptation is stale.
