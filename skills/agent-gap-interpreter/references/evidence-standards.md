# Evidence Standards

A gap needs evidence from the recorded run:

- manifest goal and stop condition
- commands run
- tests/checks and results
- review defects or corrections
- guardrail outcomes
- model/token/cost records
- compact trace events

Separate:

- evidence: directly recorded fact
- inference: reasoned explanation
- candidate correction: possible future change

Do not promote a candidate correction from one weak example unless the user
explicitly accepts it as a policy decision.
